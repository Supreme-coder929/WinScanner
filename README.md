
# WinScanner

WinScanner is a light gui-based network scanner that automatically scans everyone in your network. The data can be exported to csv. 


## Features

- Shows Private IP
- Shows MAC address



## Deployment

To deploy this project run

```bash
  # pip3 install -r requirements.txt
  # python main.py
```
